Instructions

To calculate the Confidence interval please select a two sided answer,
then select a confidence percentage, enter the sample average, the sample
size and the standard deviation the z value will appear in the box.
Press submit and the upper, lower or both values will appear in the
box labeled answer. In case the code fails, please use the values
below to determine the Z value.

The following are critical values from the Z table:

  • alpha = .10, Z = -1.28

  • alpha = .05, Z = -1.645

  • alpha = .025, Z = -1.96

  • alpha = .01, Z = -2.33

  • alpha = .005, Z = -2.575

  • alpha = .001, Z = -3.09

  • alpha = .0005, Z = -3.295

Where "alpha" = 1-(Confidence Level Percentage/100) for lower and upper bound confidence intervals and (1-(Confidence Level Percentage/100))/2 for a Two Sided interval.
