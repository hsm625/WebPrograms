Instructions

To calculate the T Distribution please select a two sided,
upper bound, or lower bound answer, then select a confidence
percentage, enter the sample average, the sample size and the
standard deviation the z value will appear in the box. Press
submit and the upper, lower or both values will appear in the
box labeled answer. In case the code fails to decide the
T value for you, you can use the table below to find it for
yourself. The "alpha" values at the top are equal to one minus
the Confidence Level percentage / 100. For a two sided interval,
divide that by two. n-1 is found by simply subtracting one from
the sample size.
